ИНСТРУКЦИЯ ПО РЕЗЕРВНОМУ КОПИРОВАНИЮ И ВОССТАНОВЛЕНИЮ

1. РЕЗЕРВНОЕ КОПИРОВАНИЕ (backup.bat)
   - Запустите файл backup.bat
   - Будет создан файл резервной копии в папке C:\backup\postgres\
   - Формат имени: product_management_ГГГГММДД_ЧЧММСС.backup

2. ВОССТАНОВЛЕНИЕ (restore.bat)
   - Запустите файл restore.bat
   - Выберите файл для восстановления из списка
   - Подтвердите операцию
   - Дождитесь завершения восстановления

3. РУЧНОЕ РЕЗЕРВНОЕ КОПИРОВАНИЕ (командная строка)
   pg_dump -U postgres -h localhost -p 5432 -F c -b -v -f "путь\к\файлу.backup" product_management

4. РУЧНОЕ ВОССТАНОВЛЕНИЕ
   pg_restore -U postgres -h localhost -d product_management -v "путь\к\файлу.backup"

5. АВТОМАТИЧЕСКОЕ РЕЗЕРВНОЕ КОПИРОВАНИЕ (Планировщик Windows)
   - Откройте "Планировщик заданий"
   - Создайте задачу с триггером (например, ежедневно в 2:00)
   - Действие: запуск backup.bat
